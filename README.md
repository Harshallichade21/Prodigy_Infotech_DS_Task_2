### Objective
Perform data cleaning and exploratory data analysis (EDA) on the Titanic dataset to identify patterns, trends, and relationships between different variables.

### Dataset
<a href="https://github.com/Harshallichade21/Prodigy_Infotech_DS_Task_2/blob/main/test.csv">Titanic Dataset</a>

### Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

### Data Cleaning
- Removed missing values from the Embarked column.
- Filled missing values in the Age column using the mean age.
- Filled missing values in the Cabin column with "Unknown".

### Exploratory Data Analysis
The following analyses were performed:
- Survival distribution of passengers
- Gender-wise survival analysis
- Passenger class vs survival
- Age distribution of passengers
- Correlation between numerical features

### Conclusion
Data cleaning and EDA helped uncover important patterns in the Titanic dataset. These insights can be useful for understanding the factors that influenced passenger survival.
